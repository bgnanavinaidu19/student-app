# Student Performance System

## Features
- Add student details
- Subject-wise grading
- CGPA calculation
- Rank system
- Search by roll number

## Technologies Used
- HTML, CSS, JavaScript
- Node.js
- MongoDB
- Docker & Kubernetes

## How to Run

1. Clone the repository
2. Install dependencies
   npm install
3. Start server
   node server.js
4. Open frontend/index.html

## Output
- Displays marks, grades, CGPA
- Shows rank and topper
